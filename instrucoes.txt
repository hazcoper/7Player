	https://www.python.org/ftp/python/3.9.9/python-3.9.9-amd64.exe

2. abrir um terminal e correr os seguintes comandos
	pip install pyautogui
	pip install Pillow

3. Descarregar VLC
	https://www.videolan.org/vlc/download-windows.html

4. Adicionar VLC as variaveis do sistema
	ver video (por fazer)

5. Configurar VLC
	meter a tocar em full screen e escolher o monitor de output
		1.jpeg
	meter para fechar automaticamente
		show settings: all
		search: play and exit
		na opcao playlist
	meter para start on top
		show settings: all
		search: always on top
		na opcao video
